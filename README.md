Кейс-задача №3
Поиск суммы отрицательных элементов между максимальным и минимальным элементами массива






Описание проекта

Данный проект представляет собой решение кейс-задачи по программированию.

Задача заключается в обработке одномерного массива чисел и вычислении суммы отрицательных элементов, расположенных между максимальным и минимальным элементами массива.

Программа написана на языке Python и реализует алгоритм поиска экстремумов массива и последующей обработки его элементов.

Условие задачи

Дан одномерный массив A размерности N.

Необходимо:

Найти максимальный элемент массива.

Найти минимальный элемент массива.

Определить элементы, расположенные между ними.

Найти сумму отрицательных элементов на этом участке массива.

Если между максимальным и минимальным элементами отсутствуют отрицательные значения, результат равен 0.

Алгоритм решения

Решение задачи выполняется следующим образом:

Пользователь вводит количество элементов массива N.

Пользователь вводит элементы массива.

Программа определяет:

максимальный элемент массива

минимальный элемент массива.

Определяются индексы максимального и минимального элементов.

Устанавливаются границы диапазона между ними.

Выполняется перебор элементов массива между этими индексами.

Если элемент меньше 0, он добавляется к сумме.

После завершения вычислений программа выводит результат.
